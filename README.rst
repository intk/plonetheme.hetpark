============
 Het Park base
============
------------------------------
 An installable theme for Plone 5
------------------------------

`Latest version on PyPI <https://pypi.python.org/pypi/plonetheme.hetpark>`_

Installation
============================
Install plonetheme.herpark by adding it to your buildout::

    [buildout]

    ...

    eggs =
        plonetheme.hetpark

and then running ``bin/buildout``
